### [Blog 1 - Introducing OpenTelemetry & PromQL support in Amazon CloudWatch](3.1-Blog1/)
This blog introduces native OpenTelemetry metrics ingestion and PromQL querying capabilities in Amazon CloudWatch. It explains how to enable OTLP ingestion, leverage automatic AWS resource enrichment, deploy CloudWatch Container Insights on Amazon EKS, and query high-cardinality infrastructure and application metrics using PromQL in CloudWatch and Amazon Managed Grafana.

### [Blog 2 - Announcing General Availability of AWS DevOps Agent](3.2-Blog2/)
This post announces the General Availability of the AWS DevOps Agent, an AI-powered operations teammate. It discusses how the agent automates incident response, proactively prevents incidents, and handles on-demand SRE tasks. The update highlights expanded capabilities, including support for Azure and on-premises environments, custom skills, code indexing, and seamless integrations with external observability tools like Datadog, Splunk, and Grafana.

### [Blog 3 - Automate AWS Systems Manager activation for hybrid-managed node registration](3.3-Blog3/)
This technical blog demonstrates how to automate the creation and management of AWS Systems Manager Hybrid Activations credentials. By leveraging AWS CloudFormation, API Gateway, AWS Lambda, and Amazon DynamoDB, the solution provides a scalable and secure way to register on-premises servers and VMs without the operational burden of manual credential rotation.
